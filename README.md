# Mercedes-Benz A-220

This repository contains signal set configurations for the Mercedes-Benz A-220, organized by model year and version. The files are structured to allow for easy differentiation between model generations and other vehicle parameters, ensuring accurate signal mapping for each version of the Mercedes-Benz A-220.

The A-220 is a trim level within the fourth generation (W177) Mercedes-Benz A-Class, featuring a 1.991L turbocharged four-cylinder engine producing 190 hp. It is available as both a hatchback and sedan, with optional 4MATIC all-wheel drive.
